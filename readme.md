# Focus Now

Focus Now 是一个更好的待办事项 Web 应用程序，它可以帮助你更有效地管理你的任务。

## 功能

- 创建、编辑和删除任务
- 拖放重新排序任务
- 标记任务为已完成
- 过滤显示所有、未完成或已完成的任务

## 安装

首先，克隆这个仓库：

```bash
git clone https://github.com/xwchris/focus-now.git
```

然后，进入项目目录并安装依赖：

```bash
cd focus-now
npm install
```

最后，启动开发服务器：
```bash
npm dev
```

现在，你应该可以在 `http://localhost:5173` 看到应用程序。

## 技术栈
- React
- Tailwind CSS
- react-beautiful-dnd
